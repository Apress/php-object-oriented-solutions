Pos_MysqlOriginalConnection

Description
Description | Vars (details) | Methods (details)

A wrapper class for connecting to MySQL with the MySQL original extension.

Apart from the constructor and destructor, this has only one method: getResultSet(), which returns a Pos_MysqlOriginalResult object.
